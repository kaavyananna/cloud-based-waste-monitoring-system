# cloud-based-waste-monitoring-system
cloud based waste monitoring system

Install ThingSpeak Library and ESP8266wifi-master

Connections

ultrasonic sensor -> NodeMcu module
trig pin  -> D0 
echo pin -> D1 
Gnd -> Gnd
Vcc -> Vin
